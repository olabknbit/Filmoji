# Filmoji
This is a simple parser, that uses https://www.themoviedb.org/ API to insert data to Neo4J database.

### Prerequisites

Install python 2.+ 
```
https://www.python.org/downloads/
```
Install neo4j
```
https://neo4j.com/download/
```
Install requests
```
$ pip install requests
```
Install py2neo
```
$ pip install py2neo
```
